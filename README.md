# CyberArena Gaming Hub

## Overview

CyberArena Gaming Hub is a responsive online gaming and esports website designed to provide gamers with a centralized platform to discover games, explore tournaments, view leaderboards, and engage with the gaming community.

The project focuses on delivering a modern gaming-themed user experience with intuitive navigation, responsive layouts, and tournament-focused functionality.

---

## Project Features

### Home Page
- Hero banner
- Featured tournaments section
- Popular games section
- Community statistics
- Call-to-action buttons
- Footer navigation

### Games Page
- Game thumbnails
- Game titles
- Category information
- Short descriptions
- Play/Register buttons

### Tournament Page
- Tournament listings
- Event dates
- Entry fee information
- Prize pool details
- Registration status
- Registration buttons

### Leaderboard Page
- Player rankings
- Usernames
- Match statistics
- Total points
- Win percentages

### Contact Page
- Name field
- Email field
- Subject field
- Message field
- Submit button

### Registration Form
- Full Name
- Gamer Tag
- Email
- Selected Tournament
- Submit Button
- Client-side validation
